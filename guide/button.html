# button按钮

# 普通按钮

#

# 代码

<bf-btn>按钮</bf-btn>

# 带icon按钮

#

icon默认位置在文字左边

# 代码

<bf-btn icon="settings">按钮</bf-btn>

# 带icon按钮（icon在右边）

#

若要将icon放置在文字右边，只需添加icon-position属性，该属性值接受left和right

# 代码

<bf-btn icon="like" icon-position="right">按钮</bf-btn>

# loading按钮

#

若要设置按钮的loading状态，只需要将loading属性值设置为true，注意该属性值为动态，需要在loading前面加上冒号:

# 代码

<bf-btn :loading="true">按钮</bf-btn>
