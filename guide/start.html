    # 使用前准备

    使用本框架前请在您的项目中进行如下步骤：

    1. 先添加css样式
    *{  box-sizing: border-box; }
    1. 添加iconfont链接
    <script src="//at.alicdn.com/t/font_1092490_1oa4felgew8.js"></script>

    # 安装beef-ui

    npm安装

    npm install beef-ui

    WARNING

    如遇样式出问题，请检查您的项目是否可以正常使用scss（sass）
