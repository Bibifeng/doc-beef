# input输入框

# 普通态&禁用态

#

# 代码

<bf-input></bf-input>
<bf-input :disabled="true"></bf-input>

# 错误提示

#

只需给输入框一个error属性即可，其中error的值为字符串

# 代码

<bf-input error="出错啦！"></bf-input>

# 事件绑定代码

#

//change事件
<bf-input @change="inputChange"></bf-input>  //inputChange为自定义事件，下同
//input事件
<bf-input @input="inputChange"></bf-input>
//focus事件
<bf-input @focus="inputChange"></bf-input>
//blur事件
<bf-input @blur="inputChange"></bf-input>
